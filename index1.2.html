Основные теги

Тег <table> </table> определяет таблицу HTML.
Все элементы таблицы находятся внутри контейнира table.
Тег <tr> </tr> определяет каждую строку таблицы.
Тег <th> </th> определяет каждый заголовок таблицы
Тег <td> </td> определяет каждую ячейку таблицы
По умолчанию текст в <th> элементах выделен полужирным шрифтом и центрирован.
По умолчанию текст в <td> элементах обычный и выровнен по левому краю.

Границы и отступы

<border> - добавляет границы в таблице. table, th, td {
border: 1px solid black;}
 border-collapse - границы сжимаются в одну границу
table, th, td {
border: 1px solid black;
border-collapse: collapse;}
padding - устанавливает отступ между содержимым ячейки и ее границами.
th, td {
padding: 15px; }
text-align - выравнивает заголовки таблицы по левому краю.
th {
text-align: left; }
border-spacing - устанавливает растояние между границами ячеек
table {
border-spacing: 5px; }
Примечание: если таблица имеет свернутые границы, border-spacing не действует.

Свойства ячейки

rowspan - делает ячейку шириной более одной строки
<table style="width:100%">
<tr>
<th>Name: </th>
<td>Bill Gates </td>
</tr>
<tr>
<th rowspan="2">Telephone: </th>
<td>55577854 </td>
</tr>
<tr>
<td>55577855 /td>
</tr>
</table>
Тег caption - добавляет заголовок к таблицы.
<caption>Monthly savings</caption> Примечание.
<caption>Тег должен быть вставлен сразу после тега<table>.
Чтобы определить специальный стиль для одной конкретной таблицы, добавьте id к таблице. table id="t01">

Границы и отступы

border-collapse - Определяет будут ли границы рамки объединены в одну.
border-spacing - Определяет расстояние между границами смежных ячеек и общей рамкой таблицы.
caption-side - Указывает расположение заголовка таблицы
empty-cells - Регулирует видимость пустых ячеек в таблице.
table-layout - Задает алгоритм использования макета таблицы.

flexbox

align-content - Задаёт вертикальное выравнивание flex-элементов для flex-контейнеров с многострочным содержимым
align-items - Вертикальное выравнивание flex-элементов внутри flex-контейнера.
align-self - Вертикальное выравнивание flex-элемента.
flex - Определяет ширину, сжатие и растяжение для flex-элемента.
flex-basis - Указывает ширину для flex-элемента.
flex-direction - Направление расположения flex-элементов.
flex-flow - Направление расположения и возможность переноса для flex-элементов.
flex-grow - Определяет коэффициент растяжения flex-элемента.
flex-shrink - Определяет коэффициент сжатия flex-элемента.
flex-wrap - Определяет возможность переноса flex-элементов.
justify-content - Горизонтальное выравнивание flex-элементов.
order - Определяет порядок вывода flex-элементов.
