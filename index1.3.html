CSS, Фон, Цвет

Цвет и Фон

Свойство color используется , чтобы установить цвет текста.
h1 { color: green; }
Цвет фона - background-color
background-color - фоновый цвет
background-color: lightblue;
background-image - изображение на заднем плане
background-image: url("paper.gif");
background-repeat - фон-повтор
background-repeat: repeat-x;
background-attachment - фоновое приложение
background-attachment: fixed;
background-position - фоновая позиция
background-position: right top;
opacity Свойство определяет непрозрачность / прозрачность элемента.
Тег <jmg> предназначен для внедрения графики на web страницы.
Атрибуты:
src - адрес изображения.
align - вид выравниваия изображения.
left - Выравнивание по левому краю.
right - Выравнивание по правому краю.
bottom - Выравнивание по нижнему краю.
top - Выравнивание по верхнему краю.
middle - Выравнивание по середине.
absmiddle - Выравнивает изображение по горизонтальной оси текстовой строки.
texttop - Выравнивание верхнего края изображения по верхнему краю текстовой строки.
absbottom - Выравнивание нижнего края изображения по самому нижнему краю текстовой строки.
alt - Выводит текст если картинка не загружена.
width - Ширина картинки
height - Высота картинки
border - Устанавливает толщину рамки вокруг изображения в пикселах.
hspace - Определяет размер свободного места в пикселах слева и справа от изображения.
vspace - Определяет размер свободного места в пикселах сверху и снизу от изображения.
title - Всплывающая подсказка
