Тег <code> </code> определяет таблицу HTML.
Все элементы таблицы находятся внутри контейнира table.
Тег <tr> </tr> определяет каждую строку таблицы.
Тег <th> </th> определяет каждый заголовок таблицы
Тег <td> </td> определяет каждую ячейку таблицы
По умолчанию текст в <th> элементах выделен полужирным шрифтом и центрирован.
По умолчанию текст в <td> элементах обычный и выровнен по левому краю.
 CSS border -добавляет границы в таблице.
table, th, td {
border: 1px solid black;}
 border-collapse - границы сжимаются в одну границу
table, th, td {
border: 1px solid black;
border-collapse: collapse;}
padding - устанавлимает отступ между содержимым ячейки и ее границами.
th, td {
padding: 15px; }
text-align - выравнивает заголовки таблицы по левому краю.
th {
text-align: left; }
border-spacing - устанавливает border-spacing.
table {
border-spacing: 5px; }
Примечание: если таблица имеет свернутые границы, border-spacing не действует.
rowspan - делает ячейку шириной более одной строки
<table style="width:100%">
<tr>
<th>Name: </th>
<td>Bill Gates </td>
</tr>
<tr>
<th rowspan="2">Telephone: </th>
<td>55577854 </td>
</tr>
<tr>
<td>55577855 /td>
</tr>
</table>
Тег caption - добавляет заголовок к таблицы.
<caption>Monthly savings</caption> Примечание.
<caption>Тег должен быть вставлен сразу после тега<table>.
Чтобы определить специальный стиль для одной конкретной таблицы, добавьте id к таблице. table id="t01"> 		colspan - ячейка занимает более одного столба

Используйте <table> элемент HTML для определения таблицы.
Используйте <tr> элемент HTML для определения строки таблицы.
Используйте <td> элемент HTML для определения данных таблицы.
Используйте <th> элемент HTML для определения заголовка таблицы.
Используйте <caption> элемент HTML для определения заголовка таблицы.
Используйте border свойство CSS для определения границы.
Используйте border-collapse свойство CSS, чтобы свернуть границы ячеек.
 Используйте padding свойство CSS, чтобы добавить отступ к ячейкам.
Используйте text-align свойство CSS для выравнивания текста в ячейке.
Используйте border-spacing свойство CSS, чтобы установить интервал между ячейками.
Используйте colspan атрибут, чтобы ячейка занимала много столбцов.
Используйте rowspan атрибут, чтобы ячейка занимала много строк.
Используйте id атрибут, чтобы однозначно определить одну таблицу.
