HTML CSS : Работа с текстом

На главную

CSS

align -выравнивает заголовок в соответствии со следующими значениями:
center - по центру.
left - по левому краю.
right - по правому краю.
font - определяет цвет, размер и выводимый шрифт.
color - определяет цвет текста.
face - определяет гарнитуру шрифта.
size - определяет размер текста в пределах от 1 до 7
column-count -разбивает текст на колонки
article {column-count: 2;}
text-transform - используется для указания прописных и строчных букв
параметры : uppercase;lowercase;capitalize;

HTML

Контейнер <i> </i>выделяет текст курсивом.
Контейнер <small> </small> выводит более мелкий, чем окружающий текст.
Контейнер <big> </big> выводит более крупный, чем окружающий текст.
Контейнер <u> </u>делает текст подчеркнутым.
Контейнер <code> </code> применяют для выделения программного кода, отображаемого на странице.

HTML

Тег <g> </g> делает текст зачеркнутым.
Тег <tt> </tt> выделяет текст моноширинным шрифтом.
Тег <kbd> </kbd> выделяет текст моноширинным шрифтом.
Тег <pre> </pre> предварительно отформатированный текст.
автоматически ставятся пробелы и переносы подходитдля написния кода
Тег <mark> </mark> маркирует текст,подсвечивается жёлтым цветом
Тег <details> </details> используется для раскрытие скрытой информации -
<summary>"Подробнее" </summary>
Тег <marquee> заставляет текст перемещаться из стороны в сторону.
Атрибуты:
behavior - Определяет вид движения.
alternate - Колебательные движения налево и направо.
scroll - Перемещение текста в направлении, указанном в direction. Достигнув края экрана, надпись появляется снова с противоположной стороны.
slide - Схоже с scroll, но текст перемещается только один раз и останавливается.
direction - Определяет направление движения.
down - Движение вниз.
left - Движение справа налево (по умолчанию).
right - Движение слева направо.
up - Движение вверх.

CSS

text-indentСвойство используется для задания отступа первой строки текста.
letter-spacingСвойство используется , чтобы указать пространство между символами в тексте.
line-heightСвойство используется , чтобы указать расстояние между строками
word-spacingСвойство используется , чтобы указать пространство между словами в тексте.
text-shadowСвойство добавляет тень к тексту.
font-sizeСвойство устанавливает размер текста.
